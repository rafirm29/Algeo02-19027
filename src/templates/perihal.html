Konsep Singkat

Pada tugas kali ini, Search Engine yang dibuat dengan metode yang memanfaat Model Ruang Vektor pada Temu-balik Informasi atau Information Retrieval. Pertama, akan dibuat array vectorizer dari seluruh kata unik yang terdapat pada input query serta seluruh dokumen terkait, sehingga akan terdapat satu array vectorizer dari input query, dan n array vectorizer dari n jumlah dokumen. Setelah itu, akan dihitung similaritas antara array vectorizer dari input query dan array vectorizer dari dokumen terkait dengan menghitung nilai dari cosine similarity. Setelah itu, dilakukan sorting berdasarkan nilai cosine similarity dari setiap dokumen. Semakin tinggi nilai dari cosine similarity tersebut, semakin tinggi similaritas dari dokumen tersebut dengan input query yang ditujukan pengguna Search Engine.


How to Use

    1. Untuk mengunggah file.txt yang sudah ada, klik tombol ‘choose file’, kemudian akan muncul window untuk memilih file yang akan diunggah. Setelah itu, tekan tombol ‘upload’ untuk mengunggah file yang sudah dipilih.
    2. Program ini juga dapat mengambil artikel dari internet kemudian menyimpannya dalam file .txt. Salin link artikel yang ingin disimpan dan tulis nama file yang ingin dibuat.
    3. Klik ‘add’. Setelah itu, akan ada file yang berisi artikel dari internet di directory kita.
    4. Untuk mencari menjalankan search engine, ketik suatu kata atau kalimat dalam form query yang sudah disediakan. Setelah itu, klik ‘search’.
    5. Hasil dari searching yang kita lakukan akan muncul. Pada laman tersebut, terdapat nama dokumen, jumlah kata, tingkat kemiripan dokumen dengan search query, dan kalimat pertama dari dokumen tersebut. Selain itu, pada bagian bawah halaman, ada tabel jumlah kata unik yang terdapat pada dokumen yang sesuai dengan search query.
    6. Jika judul dokumen tersebut diklik, akan muncul isi dari dokumen tersebut.
